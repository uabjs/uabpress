# 首页
基于 Vue3.0 SSR 的一个快速高效的 Markdown 网站制作工具

## 全局安装 uabpress 包
```
yarn add -g uabpress
```

## 运行 docs 文件夹内md文件自动编译成文档网页形式指定端口输出
```
// 在您的项目路径内
cd docs
uabpress dev
```

## 将 docs 文件夹打包成html页面
```
// 在您的项目路径内
cd docs
uabpress bulid
```

## 使用方法：
```
只需在 docs 文件夹中简单的编写您的 md 文件，即可为您编译成一个说明文档网页
  
```
