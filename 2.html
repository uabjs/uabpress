uabpress简要

一、简单说明

1. 文件目录

  • docs 根目录下的.md文件为一级目录：

  • docs 根目录下的文件夹为二级目录：

  • 以此类推在二级目录下创建的文件夹为三级目录

2. 实时预览

* 添加、修改、删除 docs 目录的文件，将在浏览器中可以实时显示修改结果

3. 支持链接

  • 链接到 二级目录

4. 自动识别标题

  • 自动识别 ”一级标题 # “ 为菜单中的链接

  • 如果文件中不存在标题，则会显示 文件路径

5. 文档自动排序

  • 文档会按“路径”进行排序

  • 同级 目录中，" README.md " 会排在第一个
